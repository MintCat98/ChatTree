Please see [CONTRIBUTORS](../.github/CONTRIBUTORS.md) to check our role and contribution.

### 2. Regular Meeting Schedule
i. 21:00 *(KST)* every Thursday
ii. 13:00 *(KST)* every Sunday

### 3. Decision Rules
#### 3-1. Task Scheduling
- All tasks are scheduled based on the [Notion documentation](https://mintcat.notion.site/koi-pangyapang?source=copy_link).

#### 3-2. Task Assignment
- All task assignments are managed through Issues and Pull Requests (PRs).
- For large-scale features, a stacked PR workflow must be followed.

#### 3-3. Commit Policy
- A commit may only be made when:
    - Running npm produces no errors, and
    - The implemented feature contains no known bugs.

#### 3-4. Code Review Policy
- All work requires a mandatory review from at least one maintainer or code reviewer before merging.
